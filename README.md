# MachineLearningModeChoiceGroup6
Python Notebook contains the data cleaning code that reads the txt file.
The data cleaning file removes missing values (-1), but if you do this for all the columns, you lose a lot of data (from respondent 2265 to 886). Therefore, we have created critical columns, which contain the variables that are most essential for our research. These are divided into several groups, such as social-demographics. It results into around 1100 rows.
There is also some experimenting with a "no opinion" group, because several respondents did not fill in the attitude statements.
